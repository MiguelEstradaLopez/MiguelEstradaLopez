## ⚡ Sobre Mí

Soy un desarrollador con mentalidad de **Arquitecto de Sistemas**. Mi enfoque combina el desarrollo de software moderno con prácticas rigurosas de seguridad (**DevSecOps**). Me apasiona el control total sobre el entorno de trabajo, desde la personalización del Kernel en **Arch Linux** hasta el despliegue de microservicios en la nube.

* 🔭 Actualmente trabajando en: **Proyectos con Rust y Arquitectura .NET**.
* 🛡️ Especialidad: **Seguridad de Aplicaciones, Automatización y Linux Hardening**.
* 🌱 Aprendiendo: **Bioingeniería y Arquitectura Cloud Avanzada (AWS)**.
* ⚙️ Workflow: **Tiling Window Manager (Hyprland/i3) + Neovim**.
