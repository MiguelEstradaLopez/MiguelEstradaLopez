## ⚡ Sobre Mí

Soy un desarrollador con mentalidad de **Arquitecto de Sistemas**. Mi enfoque combina el desarrollo de software moderno con prácticas rigurosas de seguridad (**DevSecOps**). Me apasiona el control total sobre el entorno de trabajo, desde la personalización en **Arch Linux** hasta el despliegue de soluciones Fullstack.

* 🛡️ Especialidad: **Seguridad de Aplicaciones, Automatización y Linux Hardening**.
* ⚙️ Workflow: **Tiling Window Manager (Hyprland/i3) + Neovim**, priorizando la eficiencia.
* 🌱 Aprendiendo: **Ingeniería de Seguridad (Zero Trust), Arquitectura Cloud (AWS) y Desarrollo Avanzado con .NET**.
